Local Barter & Swap PlatformA simple, privacy-first, local barter platform web app that enables users to add items or services for swap, browse available listings, and manage their own offers—all running entirely in the browser using HTML, CSS, and JavaScript.FeaturesAdd new barter listings with title, description, category, location, and optional photo URLBrowse and filter available listings in a clean, card-based gridDelete your own listingsListings auto-expire after 7 days to keep content freshFully client-side, stores data in browser localStorage—no accounts or servers neededResponsive and user-friendly design for desktop and mobileGetting StartedPrerequisitesTo use this platform, all you need is a modern web browser (Chrome, Firefox, Edge, Safari).UsageUnzip the project archiveOpen index.html in any web browserAdd your items/services to swap using the formBrowse available listings below the formDelete your listings when no longer interested or after a swapDeploymentYou can host this app on any static hosting service like GitHub Pages, Netlify, or Vercel by uploading the full contents of the unzipped folder.Deploy on GitHub PagesCreate a new public repository on GitHubUpload all files (index.html, style.css, and script.js) to the repository rootGo to repository settings -> Pages and enable GitHub Pages on the main branch, root folderAccess your live site at https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/NotesAll data exists locally in the browser's storage. Clearing browser data will delete the listings.No user accounts or authentication means listings are anonymous and ephemeral.Listings older than 7 days will be automatically removed on page load.Intended for small local communities or informal swap groups.Technologies UsedHTML5CSS3 (Flexbox, Grid)Vanilla JavaScriptBrowser localStorage for persistence
